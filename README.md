<h1 align="center">COURSE MANAGEMENT SYSTEM</h1>
<p align="center">This repository contains source code of Course Management System.</p>

## Features
- User Authentication.
- Create classes.
- Join classes.
- Post assignments/materials.
- Grade distribution.
- Mobile responsive.
